Legenda do diagrama

Cores:

  • 🟠 Núcleo Comum – CLP e acesso mobile (usado em ambos cenários)
  • 🔵 Cenário 1 – Infraestrutura própria (EC2 + Mosquitto + Node-RED + MySQL)
  • 🟢 Cenário 2 – Serviços gerenciados (AWS IoT Core + RDS/DynamoDB + MQTT Dash)

Fluxos:

  • 📡 MQTT: Comunicação entre CLP e broker (Mosquitto ou AWS IoT Core)
  • 💾 Grava: Armazenamento de dados em banco de dados
  • 📊 Atualiza: Dashboard/app recebe dados em tempo real
  • ⚡ Comando: Comando remoto para acionar LED
  • 💡 LED: Feedback visual no painel da máquina
