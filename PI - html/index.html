Visão geral do projeto
Monitoramento industrial IIoT com mobile, AWS e CLP
Contexto geral

Este Projeto Integrador III simula uma consultoria tecnológica para a XYZ Metalúrgica Integrada Ltda., uma empresa que enfrenta problemas de monitoramento e comunicação em uma célula crítica de usinagem de eixos.

A partir de uma máquina equipada com CLP, sensores de temperatura, um botão de acionamento e um LED indicador, os grupos deverão implementar uma solução IIoT em dois cenários:

  • Cenário 1: Infraestrutura própria na AWS com EC2, Mosquitto, Node-RED e MySQL;
  • Cenário 2: Serviços gerenciados na AWS com IoT Core, banco gerenciado e MQTT Dash.

O foco é integrar IoT, Mobile, Cloud, IA, Estruturas de Dados, DevOps e Gestão de Pessoas/Cultura Organizacional em uma solução única e coerente.

Objetivos centrais do projeto
  • Monitorar temperatura em tempo quase real e registrar histórico.
  • Registrar eventos por meio de um botão físico conectado ao CLP.
  • Controlar um LED na máquina via comando remoto (dashboard/app mobile).
  • Permitir visualização em celulares (operadores, supervisores, manutenção).
  • Comparar duas arquiteturas de nuvem (IaaS x PaaS) na prática.
  • Refletir sobre inclusão social, registro de ocorrências e cultura de dados na fábrica.
